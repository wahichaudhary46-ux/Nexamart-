# **App Name**: NexaMart

## Core Features:

- User Authentication: Secure sign-up and sign-in functionality using Google authentication, powered by Firebase Authentication.
- User Onboarding & Profile Management: Onboarding flow to capture user's full name, mobile number, city, and address. Allows users to view and update their profile details.
- Product Category Browsing: Browse products organized into various categories such as Electronics, Fashion, Home & Living, and Beauty, displayed with an icon and product count.
- Featured Products Display: Showcase a selection of featured products with images, names, pricing, original price (for discounts), and ratings.
- Product Search: A global search bar to allow users to find products by keywords, brands, or other relevant terms across the platform.
- Shopping Cart & Wishlist: Users can add products to a shopping cart (with item count indicator) and manage a personal wishlist.
- Personalized Product Recommendations: An AI-powered tool that provides personalized product suggestions to users based on their browsing history and profile data, visible on the dashboard.

## Style Guidelines:

- Primary Color: Deep Indigo-Purple (#3F2EC4). A bold and professional yet vibrant hue, providing strong contrast on light backgrounds.
- Background Color: Very Light Periwinkle (#F7F5FA). A nearly white shade with a subtle hint of the primary hue, ensuring a clean and modern canvas.
- Accent Color: Sky Blue (#7DC2F3). A bright and refreshing blue, chosen for its complementary nature to the primary and its ability to add a dynamic pop to interactive elements and highlights.
- Headline and body font: 'Geist' (sans-serif) for a modern, clean, and legible appearance across the application. Code font: 'Geist Mono' (monospace) for displaying any code snippets.
- Utilize a minimalist, line-based icon style throughout the application to maintain a clean and contemporary visual language. Prioritize clear and intuitive SVG icons for categories, actions, and user interface elements.
- Implement a fluid, responsive grid-based layout that adapts seamlessly to various screen sizes. Feature a sticky header for consistent navigation and centered, glassmorphic card elements for login and onboarding flows. Adopt modern card-based displays for products and categories.
- Incorporate subtle, tasteful animations for page transitions, element hovers (e.g., product image scaling, button effects), and dynamic loading indicators. Ensure smooth fade-in/slide-in effects for critical UI components to enhance user engagement.
